# NFTmaker
This is a fun and simple web application that allows you to create unique hexagon-shaped profile pictures. Perfect for poking fun at NFT owners on Twitter and beyond.

## Technologies Used
- **Frontend:** HTML, CSS
- **Backend:** JavaScript
